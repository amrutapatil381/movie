# Movie Ticket Booking System

## Overview
This is a simple movie ticket booking application built using **Streamlit**. Users can select a movie, choose a date and time, pick their seats, and get a confirmation with a unique ticket ID.

## Features
- 🎬 **Movie Selection**
- 📅 **Date & Show Timing Selection**
- 🎟 **Seat Selection (Max 5 per booking)**
- ✅ **Booking Confirmation with Ticket ID**

## Installation
To run this application, follow these steps:

### 1. Clone the Repository
```sh
git clone https://github.com/your-repo/movie-ticket-booking.git
cd movie-ticket-booking
```

### 2. Install Dependencies
```sh
pip install streamlit
```

### 3. Run the Application
```sh
streamlit run app.py
```
